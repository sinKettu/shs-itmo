# Simple HTTP Server (SHS)

Based on Python's HTTP Lib.

## Build

No build required

## Run

To get help about parameters:
```python3 server.py -h```

To run with default configs:
```python3 servert.py```

## Docker

To build container:
```sudo docker build -t your/target .```

Or get existing:
```sudo docker pull sinfox/shs:itmo```
